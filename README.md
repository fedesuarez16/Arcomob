# ArcoMob - Landing Page de Revestimientos Alistonados

Landing page de alta conversión enfocada en revestimientos alistonados.

## Tecnologías

- Next.js 14
- TypeScript
- Tailwind CSS

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura

- `app/` - Páginas y layout de Next.js
- `components/` - Componentes reutilizables
- `public/` - Archivos estáticos
